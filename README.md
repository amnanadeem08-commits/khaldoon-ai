# Khaldoon AI

Company website for **Khaldoon AI** — turn Excel files into dashboards, automate reporting, and explore AI-assisted business intelligence.

- Domain: [khaldoonai.com](https://khaldoonai.com)
- Stack: Next.js (App Router) · Vercel-ready
- Brand: navy `#1A2332` · gold `#B58E45` · cream `#F5F2EA`

## Pages

Home · Products · Excel MVP · AI DataBot · Khaldoon Trade · Services · Pricing · About · Contact · Privacy · Terms

Blog and Case Studies stay hidden until real content exists.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Contact form

Copy `.env.example` to `.env.local` and set one of:

- `FORMSPREE_FORM_ID`
- `WEB3FORMS_ACCESS_KEY`
- `RESEND_API_KEY` + `RESEND_TO_EMAIL`
- `CONTACT_EMAIL`

## Deploy

1. Push to GitHub
2. Import in Vercel
3. Set contact env vars
4. Connect `khaldoonai.com`

See `docs/WEBSITE_LAUNCH_REPORT.md` for launch status.
